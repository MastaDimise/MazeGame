# Кристаллы лабиринта

Пиксельная браузерная игра: персонаж ходит по лабиринту и собирает кристаллы.

![Pixel art maze game](https://img.shields.io/badge/стиль-pixel%20art-ffd700) ![HTML5](https://img.shields.io/badge/HTML5-Canvas-e34f26)

## Как играть

- **Управление:** стрелки ← → ↑ ↓
- **Цель:** собрать все золотые кристаллы в лабиринте
- После сбора всех кристаллов нажми «Играть снова» — появится новый лабиринт

## Запуск

Откройте файл `index.html` в браузере (Chrome, Firefox, Safari и т.д.). Никакой установки и сервера не требуется.
Либо перейди по ссылке https://mastadimise.github.io/MazeGame/

## Технологии

- HTML5 Canvas
- CSS3 (шрифт Press Start 2P, пиксельная стилистика)
- Ванильный JavaScript
- Алгоритм генерации лабиринта (рекурсивное разбиение)
